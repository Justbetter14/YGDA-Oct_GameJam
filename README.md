# YGDA-Oct_GameJam
## Description
This Game was made by members of YGDA in 2025. 

## Theme
The theme of the game jam is Transformation.

## Controls of Game
###  Jump
W / Up Arrow / Space
###  Right
D / Right Arrow
###  Left
A / Left Arrow
###  Dash
Z / Right Mouse Button
### Attack
C / Left Mouse Button
