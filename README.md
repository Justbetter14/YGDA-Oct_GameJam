# YGDA-Oct_GameJam
## Description
This is a game made by TJ students of 2025 in YGDA Game Jam 2025

## Theme
The theme of the game jam is Transformation.

## Description of the Game
Magicards is a game in which you climb through dungeons, towers, and the sky to reach the evil wizard in the sky. This game allows for the use of cards collected from enemies and use that to switch out your movement type and your attack type.

## Controls of Game
###  Jump
Space / Up Arrow
###  Right
D / Right Arrow
###  Left
A / Left Arrow
## Aim Up
W
## Aim Down
S
###  Dash
J
### Attack
K
### Use cards
First Slot: 1
Second Slot: 2
Third Slot: 3
